# Bus Routing Game
## Overview
In Bus Routing Game (actual name pending), you are a city planner trying to maximize the usage of public transportation, particularly with buses.

## Gameplay
The primary goal in a city is to connect residences to the businesses of the same color using bus routes. The player can place bus stops and create routes with them. 

## Installation
A Windows exe will be available after an MVP is establshed. For now, the project can be run by importing into [Godot Engine - .NET](https://godotengine.org/download).
